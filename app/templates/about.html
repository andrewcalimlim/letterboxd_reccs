About ROLF

ROLF is a recommendation algorithm for Letterboxd users to get film recommendations. It uses linear algebra to predict user film ratings based off of other Letterboxd user ratings. The top 5-highest ratings are then displayed.

Because Letterboxd's API is invite-only, Letterboxd usernames have to be manually entered and XML files of users are parsed. Therefore only the most recent 50 entries of any user are considered, and films seen before may be recommended to a user.

For further information, check out this PDF (Note: ROLF was originally an academic project).
